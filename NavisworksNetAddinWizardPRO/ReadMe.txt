Navisworks .NET Addin Wizard Pro v2.0.4.1

Launch Visual Studio 2015/2013/2012;

Follow the following path to create a new Navisworks .NET addin project:
	New Project 
		-> Visual C#/Basic/C++
			-> Navisworks .NET Addin Pro
Choose options in wizard dialogs;
Click the Finish button on the last page.
